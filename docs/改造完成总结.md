## 🎯 核心改进

### 1. 节点设计

| 节点类型 | 主键 | 关键属性 | 说明 |
|----------|------|----------|------|
| **Repo** | name | path, git_url, branch, scanned_at | 新增: git 仓库节点 |
| **Type** | fqn | has_injection, arch_layer | 新增属性: 是否有注入字段、架构层次 |
| **Method** | signature | is_entry | 新增属性: 是否为入口方法 |
| **Field** | signature | injection_type | 新增属性: 注入类型 (仅注入字段) |
| **RpcEndpoint** | path + method | http_method, service_name | 新增: RPC/HTTP 入口节点 |
| **Job** | fqn | job_type, cron_expr | 新增: 定时/延时任务节点 |

### 2. 边设计

| 边类型 | 起点 | 终点 | 属性 | 说明 |
|--------|------|------|------|------|
| **CALLS** | Method | Method | via_field | 改进: 方法级调用 (仅注入) |
| **DUBBO_CALLS** | Type | Type | field_name, method_name | 改进: 添加 method_name |
| **EXPOSES** | Method | RpcEndpoint | - | 新增: 方法暴露 RPC 入口 |
| **EXECUTES_JOB** | Method | Job | - | 新增: 方法执行任务 |
| **ENTITY_FOR_TABLE** | Type | Table | - | 新增: Entity 映射表 |

### 3. 核心逻辑

#### 注入字段识别

```python
INJECTION_ANNOTATIONS = {
    'Reference',      # Dubbo 远程调用
    'DubboReference', # Dubbo 远程调用
    'Resource',       # 内部依赖注入
    'Autowired'       # 内部依赖注入
}

# 数据结构
injected_fields = {
    'class_fqn': {
        'field_name': {
            'annotation': 'Reference',
            'type_fqn': 'com.example.Service'
        }
    }
}
```

#### 业务类过滤

```python
# 保留条件 (满足任一)
if (has_injection or 
    is_injected or 
    is_mapper or 
    is_dubbo_interface or 
    is_entity):
    # 保留
else:
    # 过滤: Util, DTO, Request, Response, Code 等
```

#### 调用关系提取

```python
# 仅记录注入字段发起的调用
if qualifier in injected_fields[current_class]:
    field_info = injected_fields[current_class][qualifier]
    
    if field_info['annotation'] in ['Reference', 'DubboReference']:
        # Type -[DUBBO_CALLS]-> Type
    elif field_info['annotation'] in ['Resource', 'Autowired']:
        # Method -[CALLS]-> Method
```

#### arch_layer 推断

```python
# Controller
if @Controller or @RestController or @DubboService:
    return 'Controller'

# Service
if @Service (非 Dubbo):
    return 'Service'

# Manager
if class_name.endswith('Manager'):
    return 'Manager'

# DAO
if @Repository or 'dao' in package:
    return 'DAO'

# Mapper
if @Mapper or class_name.endswith('Mapper'):
    return 'Mapper'

# Entity
if @Entity or 'entity' in package:
    return 'Entity'
```

## 🔍 开通贵族示例

### 调用链结构

```
[RpcEndpoint: /official/open/noble]
  ↑ EXPOSES
[Method: NobleController.openNoble] (official-room-pro-web)
  ↓ DUBBO_CALLS (via nobleRemoteService @Reference)
[Type: NobleRemoteService] (interface)
  ↑ DUBBO_PROVIDES
[Type: NobleController] (yuer-chatroom-service @DubboService)
  ↓ DECLARES
[Method: NobleController.openNoble]
  ↓ CALLS (via nobleManager @Resource)
[Method: NobleManager.changeNobleInfo]
  ↓ CALLS (via nobleService @Resource)
[Method: NobleService.changeNoble]
  ↓ CALLS
[Method: NobleMapper.update]
  ↓ MAPPER_FOR_TABLE
[Table: chatroom_noble_info]
```

### 查询 Cypher

```cypher
MATCH (endpoint:RpcEndpoint {path: '/official/open/noble'})
MATCH (endpoint)<-[:EXPOSES]-(entry:Method)
MATCH (entry)<-[:DECLARES]-(web_controller:Type)
OPTIONAL MATCH (web_controller)-[:DUBBO_CALLS]->(dubbo_iface:Type)
OPTIONAL MATCH (dubbo_impl:Type)-[:DUBBO_PROVIDES]->(dubbo_iface)
MATCH (dubbo_impl)-[:DECLARES]->(m:Method)
MATCH path = (m)-[:CALLS*0..10]->(end:Method)
OPTIONAL MATCH (end)<-[:DECLARES]-(mapper:Type)-[:MAPPER_FOR_TABLE]->(table:Table)
RETURN path, table
```

## 🚀 使用方式

### 1. 基本使用

```python
from src.parsers.java.scanner_v2 import JavaASTScannerV2
from src.parsers.java.config import get_java_parser_config
from src.storage.neo4j.storage import Neo4jStorage

# 初始化
config = get_java_parser_config()
client = Neo4jStorage(uri="bolt://localhost:7687", user="neo4j", password="password")
scanner = JavaASTScannerV2(config=config, client=client)

# 扫描
result = scanner.scan_project(
    project_name="yuer-chatroom-service",
    project_path="d:/cursor/code-ast-graph/git-repos/yuer-chatroom-service",
    force_rescan=True
)
```

### 2. 批量扫描

```python
projects = [

    {'name': 'official-room-pro-web', 'path': '...'},
    {'name': 'official-room-pro-service', 'path': '...'}
]

for proj in projects:
    result = scanner.scan_project(proj['name'], proj['path'])
    print(f"{proj['name']}: {result['success']}")
```
